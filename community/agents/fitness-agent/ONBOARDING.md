# Onboarding

Run `/setup` on first boot by reading `.claude/skills/setup/SKILL.md`.

Setup configures:

- user goals, baseline, constraints, professional guidance, and no-go topics
- opt-in coaching tone and intensity
- tracking sources and local file source of truth
- morning plan, evening review, weekly review, and heartbeat crons
- `fitness/profile.json`, starter plan, starter check-in, logs, weekly review template, goals, memory, heartbeat, events, and `.onboarded`

Do not begin normal fitness coaching until `.onboarded` exists.
