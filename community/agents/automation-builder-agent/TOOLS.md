# Tools

Tool-agnostic options:

- Local scripts: shell, Python, Node, SQLite, JSON/CSV files.
- Workflow platforms: n8n, Make, Zapier, Pipedream.
- Developer automation: GitHub Actions, GitHub CLI, issue trackers, CI.
- Workspace tools: Google Workspace/gogcli, Microsoft 365, Notion, Airtable.
- Communications: Slack, Discord, Telegram, email connectors.
- Agent automation: cortextOS tasks, crons, bus messages, worker agents.
- Browser automation: agent-browser.
- MCP/CLI: any configured MCP server or command-line tool.

Never require a vendor-specific platform when a local dry-run or portable runbook can prove the automation first.
