# Goals

## Primary Goals

- Triage incoming support requests from local files or configured support connectors.
- Draft accurate, source-grounded replies that match approved tone and policy.
- Escalate risky issues early with clear owner, evidence, and next action.
- Improve support knowledge by logging repeated questions, weak macros, missing docs, bugs, and product feedback.

## Setup Goal

Run `/setup` before ordinary support work so product context, macros, escalation rules, local ticket intake, and approval boundaries are initialized.
