# Tool Connections

| Domain | Options |
|---|---|
| Tickets | Local `support/tickets.jsonl`, Zendesk, Intercom, Freshdesk, Help Scout, email/chat exports |
| Email | Gmail, Outlook, local `.eml` or CSV exports |
| Docs | Local docs, website exports, KB collections, Drive/Notion exports, knowledge-base librarian |
| Bugs/Product | GitHub Issues, Linear, Jira, configured product/engineering owner |
| Customer context | Local records, configured CRM exports |
| Approval | cortextOS approvals before any customer send or external ticket mutation |
