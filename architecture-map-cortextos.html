Critical path (the seam)

CLI (cortextos start | bus <verb>) → daemon (AgentManager spins up AgentProcess + FastChecker per agent) → daemon injects Telegram + inbox messages into Claude Code PTY via TTY keystrokes → Claude Code calls bus modules → atomic writes to ~/.cortextos/state/ (file locks per op). The file bus IS the central nervous system.

Top 3 surprises

  • Context-rotation circuit breaker — fast-checker.ts has a 71-hour context monitor; 3 restarts in 15 min triggers a cooldown to prevent restart loops.
  • Defense-in-depth on bus inputs — agent-name regex validation (path-traversal guard), HMAC-SHA256 message signing (optional, per-instance key), credential pattern scrub before any append to events.jsonl.
  • 30-second manual-fire cooldown — IPC handleFireCron blocks dashboard cron-spam per (agent, cron) tuple, so the dashboard can't accidentally hammer a flaky agent.
